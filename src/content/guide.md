# 📘 Panduan Pengguna: CMS Nafasz Global
**Indo-Pacific Collaborative Corridor Initiative**

Panduan ini akan membantu Anda mengelola konten website, mulai dari mengubah teks di halaman utama hingga menambahkan halaman baru ke dalam menu navigasi.

---

## 1. Cara Akses & Login
Dashboard pengelolaan website (CMS) dapat diakses melalui alamat:
👉 **[nafaszglobal.com/admin/](https://nafaszglobal.com/admin/)**

1. Klik tombol **"Login with GitHub"**.
2. Masukkan akun GitHub Anda (yang sudah memiliki akses ke repositori proyek).
3. Anda akan diarahkan ke dashboard utama.

---

## 2. Mengedit Konten Halaman Utama
Gunakan menu ini untuk mengubah teks yang sudah ada (Header, Visi, Misi, Footer).

1. Pada menu kiri, pilih **"🌐 Site Content"**.
2. Pilih salah satu:
   * **✏️ English Content**: Untuk mengedit teks bahasa Inggris.
   * **Indonesian Content (Bahasa)**: Untuk mengedit teks bahasa Indonesia.
3. Klik pada bagian yang ingin diubah (misal: *Hero Section* atau *Strategy Pillars*).
4. Ubah teks sesuai keinginan.
5. Klik tombol **"Publish"** di bagian atas.
6. **Tunggu ~2 menit** agar "Robot" mengirim perubahan ke website.

---

## 3. Menambahkan Halaman Baru
Anda bisa membuat halaman tambahan (seperti halaman *About*, *Partners*, atau *News*).

1. Pilih menu **"📄 Pages"** di sebelah kiri.
2. Klik tombol **"New Page"** (di pojok kanan atas).
3. Isi data halaman:
   * **Title**: Judul halaman (Inggris).
   * **Title (Indonesian)**: Judul halaman (Indonesia).
   * **URL Slug**: Alamat unik halaman (misal: `about-us`). Alamatnya akan menjadi `nafaszglobal.com/pages/about-us/`.
   * **Body**: Isi konten halaman (mendukung format tulisan tebal, miring, link, dll.).
4. **Pengaturan Penting**:
   * **Published**: Geser ke **ON** agar halaman bisa diakses publik.
   * **Show in Menu**: Geser ke **ON** jika Anda ingin judul halaman ini muncul di tombol menu (Hamburger).
   * **Menu Order**: Masukkan angka (1, 2, 3...) untuk menentukan urutan di menu.
5. Klik **"Publish"**.

---

## 4. Mengelola Menu Navigasi (Hamburger)
Tombol garis tiga (≡) di pojok kanan atas bisa diatur secara global.

1. Pilih menu **"⚙️ Settings"** -> **"🍔 Menu & Navigation"**.
2. **Enable Navigation Menu**:
   * **ON**: Tombol menu akan muncul di website.
   * **OFF**: Tombol menu akan hilang sama sekali.
3. Klik **"Publish"**.
   * *Catatan: Perubahan ini bersifat instan (tidak perlu menunggu Robot).*

---

## 5. Tips Penting & Troubleshooting

### ⏳ Berapa lama perubahan muncul?
Setiap kali Anda menekan **Publish** (kecuali pada menu Settings), diperlukan waktu sekitar **2 menit** bagi sistem "Robot" kami untuk merakit ulang website dan mengirimnya ke server. Mohon bersabar sejenak sebelum melakukan *refresh* browser.

### 🔄 Perubahan tidak muncul?
Jika sudah menunggu 2 menit tapi belum berubah:
1. Lakukan **Hard Refresh** (Tekan `Cmd + Shift + R` di Mac atau `Ctrl + F5` di Windows).
2. Pastikan status di CMS sudah **"Published"** (bukan "Draft").

### 🖼️ Mengunggah Gambar
Saat mengedit halaman, Anda bisa mengunggah gambar melalui field **"Featured Image"**. Pastikan ukuran file tidak terlalu besar (disarankan di bawah 500KB) agar website tetap cepat diakses.

---

> [!TIP]
> **Selalu simpan perubahan Anda!** Jika Anda keluar dari halaman tanpa menekan tombol "Publish", perubahan Anda mungkin akan hilang.
